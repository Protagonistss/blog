# 一、安装

## Linux、Macos

```bash
curl --proto '=https' --tlsv1.2 https://sh.rustup.rs -sSf | sh
```

# 二、查看版本

`rustc -v`

`cargo -v`

# 三、卸载

`rustup self uninstall`

# 四、更新

`rustup update`

# 五、vscode插件

- Better TOML
- Error Lens 展示错误信息
- CodeLLDB, Debugger
